# TMarks - 智能书签管理与新标签页扩展

## 🌟 核心优势

### 1. AI 智能书签管理
- **AI 自动标签推荐**：使用 AI 分析网页内容，自动生成相关标签
- **智能分类**：AI 帮助你快速整理和分类书签
- **一键保存**：浏览器扩展一键快捷存储书签，支持快捷键操作
- **封面图自动获取**：自动抓取网页封面图，让书签更直观
- **网页快照**：支持保存完整网页快照，离线也能查看

### 2. 标签页收纳功能
- **一键收纳浏览器**：打开了很多资料网页不想关闭？一键收纳所有标签页
- **临时保存**：不想加入书签的网页，可以临时收纳，改天再看
- **批量管理**：支持批量收纳、恢复、删除标签页
- **智能分组**：自动按窗口或主题分组收纳的标签页

### 3. 美观实用的新标签页
- **置顶书签 Dock 栏**：最常用的书签固定在底部 Dock 栏，一键访问
- **Bing 每日壁纸**：
  - 自动获取 Bing 每日精美壁纸
  - 支持查看过去 7 天的历史壁纸
  - 手动刷新获取最新图片
  - 显示图片信息（标题、版权、日期）
- **快捷方式网格**：自定义快捷方式，支持拖拽排序
- **智能搜索**：集成多个搜索引擎，支持搜索建议
- **实用组件**：时钟、天气、待办事项、热搜榜等

## 🎯 使用场景

### 场景 1：研究资料收集
你正在研究某个主题，打开了 20+ 个网页：
1. 使用 **一键收纳** 功能保存所有标签页
2. AI 自动分析并添加相关标签
3. 改天继续研究时，快速恢复所有标签页

### 场景 2：日常浏览
浏览网页时发现有用内容：
1. 点击扩展图标或使用快捷键
2. AI 自动推荐标签，一键保存
3. 重要书签可以置顶到新标签页 Dock 栏

### 场景 3：高效工作
每天打开浏览器：
1. 新标签页显示 Bing 精美壁纸
2. Dock 栏展示常用工作网站
3. 快捷方式网格放置项目相关链接
4. 搜索框快速查找资料

- **React 18** + **TypeScript**：类型安全的现代开发
- **Tailwind CSS**：快速样式开发
- **Zustand**：轻量级状态管理
- **dnd-kit**：流畅的拖拽体验

### 后端技术栈
- **Cloudflare Workers**：边缘计算，全球加速

## 🚀 快速开始

### 安装扩展
1. 下载对应浏览器的扩展包
2. 打开浏览器扩展管理页面
3. 启用"开发者模式"
4. 加载解压后的扩展

### 配置同步（可选）
1. 部署 TMarks 后端到 Cloudflare
2. 在扩展设置中配置 API 地址和密钥
3. 开始享受云端同步
